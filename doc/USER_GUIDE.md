## Introduction

### Qu'est-ce qu'EcoTri ?

EcoTri est une application mobile innovante conçue pour simplifier et améliorer vos pratiques de recyclage au quotidien. Développée en collaboration avec Bordeaux Métropole, cette application vous accompagne dans la gestion responsable de vos déchets en vous fournissant des informations précises sur les points de collecte, les horaires de ramassage et les bonnes pratiques de recyclage.

### Objectifs de l'Application

- **Simplifier le recyclage** : Accès facile aux informations de collecte
- **Améliorer la précision** : Reconnaissance d'objets par intelligence artificielle
- **Localiser les points de collecte** : Géolocalisation des centres de recyclage
- **Suivre vos progrès** : Statistiques personnalisées de recyclage
- **Contribuer à l'environnement** : Impact positif sur la planète

### Compatibilité

- **iOS** : Version 12.0 et supérieures
- **Android** : Version 8.0 (API 26) et supérieures
- **Taille minimale** : 100 MB d'espace libre
- **Connexion** : Internet requis pour les fonctionnalités principales

## Gestion des Déchets

### Scanner de Déchets

#### Comment Scanner un Déchet

1. **Accédez au scanner**

   - Appuyez sur le bouton "Scanner un déchet" depuis l'accueil
   - Ou utilisez l'onglet "Scanner" dans la navigation

2. **Préparez votre déchet**

   - Placez l'objet sur une surface plane et bien éclairée
   - Assurez-vous que l'objet est entièrement visible
   - Évitez les reflets et ombres excessifs

3. **Effectuez le scan**
   - Pointez la caméra vers l'objet
   - Maintenez l'appareil stable
   - Appuyez sur le bouton de capture

#### Conseils pour un Scan Réussi

- **Éclairage optimal** : Utilisez une lumière naturelle ou artificielle uniforme
- **Cadrage précis** : L'objet doit occuper 70% de l'image
- **Stabilité** : Évitez les mouvements pendant la capture
- **Netteté** : Assurez-vous que l'image est nette et non floue
- **Arrière-plan neutre** : Utilisez un fond uni et contrasté

#### Résultats du Scan

Après le scan, vous obtenez :

- **Identification de l'objet** : Nom et catégorie du déchet
- **Conseils de recyclage** : Instructions spécifiques au type de déchet
- **Points de collecte** : Localisation des centres de recyclage appropriés (à venir)
- **Impact environnemental** : Informations sur l'empreinte carbone
- **Alternatives durables** : Suggestions de réutilisation ou de réduction

### Gestion des Types de Déchets

EcoTri prend en charge tous les types de déchets courants :

- **Verre** : Bouteilles, pots, bocaux
- **Plastique** : Emballages, bouteilles, films
- **Papier** : Cartons, journaux, magazines
- **Métal** : Canettes, conserves, emballages
- **Électronique** : Piles, petits appareils
- **Textile** : Vêtements, tissus
- **Organique** : Déchets alimentaires, compost

## Conseils et Astuces

### Guide de Recyclage

#### Bonnes Pratiques

- **Tri sélectif** : Respectez les consignes de tri
- **Nettoyage** : Rincez les emballages avant recyclage
- **Compactage** : Réduisez le volume des déchets
- **Éviter la contamination** : Ne mélangez pas les types de déchets

#### Conseils par Type de Déchet

- **Verre** : Retirez les bouchons et couvercles
- **Plastique** : Vérifiez le code de recyclage
- **Papier** : Évitez les papiers souillés ou gras
- **Métal** : Videz complètement les contenants
- **Électronique** : Utilisez les points de collecte spécialisés

## Annexes

### Glossaire

- **Collecte sélective** : Tri des déchets par type
- **Point de collecte** : Lieu de dépôt des déchets
- **Recyclage** : Transformation des déchets en nouvelles matières
- **Tri sélectif** : Séparation des déchets selon leur nature
- **Valorisation** : Réutilisation des déchets comme ressources
